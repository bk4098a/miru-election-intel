01 — 배경 · 목적

왜 지금 필요한가

MIRU는 한국·필리핀·키르기스스탄·콩고·이라크·파라과이 6개국에 투표·개표 장비를 공급하고 있습니다. 각국 조달 포털은 언어·형식·접근 방식이 제각각이라 담당자가 수동으로 모니터링하는 데 한계가 있었습니다.

🌍
35개국 커버리지
DRE · EVM · OMR 방식의 기계투표 도입 국가 전수 조사 완료. 71개 포털 식별 및 크롤 가능성 분석.
🎯
6개국 핵심 시장
MIRU 납품 이력 국가(KOR·PHL·KGZ·COD·IRQ·PRY) 우선 모니터링. 재계약·갱신 타이밍 포착.
⚡
실시간 알림
경쟁사(Smartmatic 등) 동향 추적. 입찰 공고 게시 즉시 감지 → 마감 전 대응 가능.
🤖
완전 자동화
GitHub Actions 일일 크롤 → DB 업데이트 → 웹 공개. 수동 작업 없이 최신 데이터 유지.
election-monitor와의 역할 분리: 기존 election-monitor repo(119개 포털 + Streamlit 대시보드)와 분리 운영. 본 프로젝트는 기계투표(DRE/EVM/OMR) 35개국에 집중한 전략 인텔리전스 레이어.

02 — 문제 정의

분산된 정보, 놓치는 기회

71개 포털이 35개국에 분산되어 있고, 언어·접근 방식·공개 수준이 제각각입니다. 수동 모니터링으로는 한계가 있었습니다.

① 분산

71개 포털 × 35개국 × 정부조달·선관위·EMB 유형 혼재. 하나의 창구로 볼 수 없음.

② 이질성

아랍어·스페인어·몽골어 등 다국어. REST API · HTML · SPA · 로그인 필요 등 접근 방식 다양.

③ 적시성

수동 확인 지연 → 마감 후 발견. Kenya 463건처럼 포털에 있는 공고를 실시간으로 포착하지 못함.

④ 경쟁 가시성

Smartmatic·Dominion의 계약 갱신 시점 파악 불가. MIRU 납품국의 재입찰 주기 모니터링 체계 없음.

03 — 솔루션 A · 완료

Election Procurement Radar

지도 중심의 단일 인텔리전스 대시보드. 206개국 투표 방식·공급사 현황을 한눈에 파악하고, 경쟁사 포지셔닝과 계약 기회를 분석합니다.

FEATURE 01
글로벌 지도 뷰
35개국 투표 방식별 색상 구분. MIRU 납품국 빨간 링 하이라이트. 줌·드래그·클릭 드릴다운. Standalone HTML — 오프라인 사용 가능.
FEATURE 02
포털 디렉토리
국가 → 포털 목록 → 상세 정보. 유형·언어·수집 방법·비고 표시. 실제 사이트 링크 연결. 71개 포털 전수 수록.
FEATURE 03
통계 드릴다운
방식·지역·공급사·우선순위별 분포. 막대 클릭 → 해당 포털 필터링. Chart.js 인터랙티브 차트.
FEATURE 04 완료
경쟁사 탭
Smartmatic·Dominion 등 주요 경쟁사 포털 커버리지, 지역별 점유, H2H 비교. 클릭 시 상세 슬라이드 패널.
FEATURE 05 완료
계약·기회 탭
MIRU 기존 계약 만료 연도별 버킷. 2026–2027 재계약 임박 국가 하이라이트. 클릭 → 지도 연동.
FEATURE 06
다크/라이트 모드
localStorage 연동 테마 전환. 미국 주별 타일맵 포함. Standalone HTML — CDN 없이 완전 동작.
